# Leyton CognitX — Open Source

**[Leyton CognitX](https://cognitx.leyton.com/)** is the AI & technology advisory of Leyton, uniting **200+ tech builders and data scientists** across **17 countries**. We build cutting-edge AI and tech solutions — from strategic advisory to custom development.

This is our open source portal. Explore our tools below.

---

## Our Projects

### [Clevexport](https://github.com/cognitx-leyton/clevexport)
**Async Laravel Excel export for large datasets, chunked & queued**

Export thousands of rows to Excel without crashes or timeouts. Uses Laravel queues to chunk large datasets into manageable sub-exports and merge them reliably.

`PHP` `Laravel` `Excel` `Queue Jobs`

---

### [TS OData V4 Server](https://github.com/cognitx-leyton/ts-odata-v4-server)
**TypeScript OData V4 server with decorators, streaming, and async support**

Full-featured OASIS Standard OData V4 server for Node.js. Supports filtering, sorting, paging, projection, expansion, CRUD operations, streaming, and media entities — all configured via TypeScript decorators.

`TypeScript` `Node.js` `Express` `OData`

---

### [Vue Input Highlighter](https://github.com/cognitx-leyton/vue-input-highlighter)
**Real-time regex highlighting for Vue 3 inputs with cursor preservation**

A Vue 3 component that highlights content inside input fields based on regular expressions. Perfect for search operators, ID validation, syntax highlighting, and more.

`Vue 3` `TypeScript` `Regex` `Contenteditable`

---

### [Laravel Circuit Breaker](https://github.com/cognitx-leyton/laravel-circuit-breaker)
**Laravel circuit breaker with configurable thresholds and cache drivers**

Prevent cascading failures in distributed systems. Implements the Circuit Breaker pattern (Closed/Open/Half-Open) with multi-service support, cache-driven state management, and configurable failure thresholds.

`PHP` `Laravel` `Redis` `Microservices`

---

### [PostgreSQL OData Server](https://github.com/cognitx-leyton/pg-odata-server)
**OData V4 server for PostgreSQL with TypeScript decorators**

Expose PostgreSQL tables as OData V4 REST endpoints. Built on ts-odata-v4-server with full query capabilities — filtering, expansion, and relationship support.

`TypeScript` `PostgreSQL` `Node.js` `OData`
